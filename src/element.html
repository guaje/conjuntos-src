## Elemento

Elemento: Es cualquier objeto para el cual el predicado del conjunto es verdadero.

e.g. Sea el conjunto \(A\), se dice que x es un elemento de A si \(\phi_A(x) = V\). En el caso en que \(\phi_A(x) = F\), se dice que el objeto x no es un elemento de A.

Elemento: Ejemplo

Para el conjunto \(A = \{1, 2, 3, 4\}\), se tiene que 1 es elemento de A, pues (\(\phi_A(1) = V\)), y 5 no es elemento de A, pues (\(\phi_A(5) = F\)).
