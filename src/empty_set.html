## Conjunto vacío

Conjunto vacío: Definición

Conjunto vacío: Es el conjunto para el cual se cumple el predicado \(\phi_\emptyset(x) = \{x : (x \neq x)\}\). Es decir, que no tiene elemento alguno porque no existe objeto alguno que sea diferente de si mismo.

  • El conjunto vacío se nota como \(\emptyset\) o \(\{ \}\).
  • Para todo objeto \(x\) se tiene que \(\phi_\emptyset(x) = F\).

Conjunto vacío: Consideraciones

Es importante tener en cuenta que \(\{ \} = \emptyset \neq \{\emptyset\}\), pues este último es el conjunto que tiene un único elemento que es el conjunto vacío, por lo tanto no es un conjunto vacío.
