## Conjunto

Conjunto: Definición

\(A = \{"Rojo", "Verde", "Azul"\}\)

Conjunto: Colección bien definida de objetos.

Conjunto: Constructor

$$\phi_{Stark}(x) = \{x : x~\text{es un miembro de la familia Stark}\}$$

Contructor del conjunto: Es el predicado que determina de manera exacta los objetos que pertenecen a un conjunto.

  • El construcctor es particular para cada conjunto, es decir, existe un constructor para el conjunto \(A\), otro para el conjunto \(B\), etc.
  • El constructor de un conjunto particular se nota de la siguiente manera: \(\phi_{NombreDelConjunto}\).

Conjunto: Ejemplo

La colección \(A = \{1, 2, 3, 4\}\) es un conjunto, ya que el siguiente predicado determina de manera exacta los objetos que pertenecen a \(A\):

$$􏰄\phi_A(x) = \begin{cases} V, & \text{si $x = 1$, $x = 2$, $x = 3$ o $x = 4;$} \\ F, & \text{en otro caso.}\end{cases}$$
