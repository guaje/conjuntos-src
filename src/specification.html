## Especificación
## Especificación: Introducción Gracias a las definiciones anteriores, un conjunto se puede especificar de dos maneras: por _extensión_ y por _comprensión_.

Especificación: Extensión

La notación general de un conjunto especificado por extensión es: $$A = \{x_1, x_2, \dots ,x_n\}$$ donde \(x_i\) son los objetos en el conjunto \(A\).

De esta manera el predicado asociado al conjunto es: $$\phi_A(x) = \begin{cases} V, & \text{si $x = x_i$ para algún $i = 1, 2, \dots, n$;} \\ F, & \text{en otro caso.} \end{cases}$$

Extensión: Ejemplos

  • \(A = \{1, 2, 3, 4\}\)
  • \(B = \{\color{blue}{\heartsuit}, \color{blue}{\diamondsuit}, \color{black}{\clubsuit}, \color{black}{\spadesuit}\}\)
  • \(V = \{'a', 'e', 'i', 'o', 'u'\}\)
  • \(\mathbb{B} = \{V, F\}\)
  • \(\mathbb{N} = \{0, 1, 2, 3, 4, 5, \dots\}\)
  • \(\mathbb{P} = \{1, 2, 3, 5, \dots\}\)
  • \(\mathbb{Z} = \{\dots, -3, -2, -1, 0, 1, 2, 3, \dots\}\)

Especificación: Comprensión

La notación general de un conjunto especificado por comprensión es: $$A = 􏰂\{x : \phi_A(x)􏰃\}$$

En esta notación, el símbolo “:” se lee “tal que”.

Compresión: Ejemplos

  • \(2\mathbb{N} = \{x : (x = 2n) \land (n \in \mathbb{N})\}\)
  • \(\mathbb{Q} = \{x : (x = p / q) \land (p, q \in \mathbb{Z}) \land (q \neq 0)\}\)
  • \(\mathbb{I} = \{x : x~\text{es un número irracional}\}\)
  • \(\mathbb{R}^+ = \{x : (x \in \mathbb{R}) \land (x \gt 0)\}\)
  • \(A = \{x : (x \in \mathbb{R}) \land (x^2 \le 1)\}\)
  • \(B = \{x : x~\text{es una pinta del poker}\}\)
  • \(V = \{x : x~\text{es una vocal}\}\)
