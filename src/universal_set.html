## Conjunto universal

Conjunto universal: Definición

Conjunto universal: Es un conjunto que contiene varios conjuntos. Es decir, un conjunto \(U\) se dice un universo para los conjuntos \(A_1, A_2, \dots, A_n\) si y sólo si \((\forall i = 1, 2, \dots, n) (A_i \subseteq U)\).

Los universos no son únicos, es decir, se pueden construir diferentes universos para una misma familia de conjuntos.

Conjunto universal: Ejemplo

Dados los conjuntos \(A = \{\lor, \land, \rightarrow, \leftrightarrow\}\), \(B = \{\lnot, \lor, \land, \rightarrow, \leftrightarrow, \Leftrightarrow, \Rightarrow\}\) y \(C = \{\lnot, \lor, \land, \rightarrow, \leftrightarrow, \Leftrightarrow, \Rightarrow, \forall, \exists\}\), se tiene que:

  • \(B\) es un universo para \(A\)
  • \(C\) es un universo para \(A\)
  • \(C\) es un universo para \(B\)
  • por lo tanto \(C\) es un universo para \(A\) y \(B\)
